## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files and folders from this project.
3. Open the repository's **Settings**.
4. Select **Pages** from the left menu.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Click **Save**.
8. After GitHub publishes the site, copy the GitHub Pages URL and submit it to the assignment.

## Replace the Temporary Profile Photo

Keep the same filename:

```text
assets/Profile_pic.png
```

Replace the temporary image with your actual photo. Because the filename stays the same, no HTML changes are needed.

## Update the Resume

Replace:

```text
assets/Rachana_S_Developer_Resume.pdf
```

with a newer PDF using the same filename.
